The Psycho-Orientation module enables counselors (psicoorientadores) to manage student referrals, create detailed psychological reports, and oversee student interventions including PIAR and DUA processes.

## Overview

Counselors are assigned to specific academic grades and receive referrals from teachers for students in those grades. The module provides comprehensive tools for:
- Reviewing and processing student referrals
- Creating psychological inquiry reports
- Managing student states (Active, PIAR, DUA)
- Tracking student history and interventions
- Uploading supporting documentation

<Note>
  Counselors only see students in the grades assigned to them during user creation. This ensures proper workload distribution and specialization.
</Note>

## Creating Psychological Reports

After reviewing a referral, counselors create a detailed psychological inquiry report.

<Steps>
  <Step title="Access Report Form">
    From the referral details page, click **Create Report** to begin.
    
    ```php
    // PsicoController.php:199-213
    public function report_student(string $id)
    {
        $groups  = Group::orderByRaw('CAST(`group` AS UNSIGNED)')->get();
        $degrees = Degree::orderByRaw('CAST(`degree` AS UNSIGNED)')->get();
        $states  = State::whereIn('state', ['activo', 'en PIAR', 'en DUA'])->get();

        $info_student = Users_student::findOrFail($id);

        return view('psycho.addReportStudent', compact(
            'groups',
            'degrees',
            'states',
            'info_student'
        ));
    }
    ```
  </Step>

  <Step title="Complete Report Fields">
    
    ### Student Information
    Verify and update:
    - Name, last name, document number
    - Age, grade, and group
    - New state (Activo, en PIAR, en DUA)
    
    ### Report Content
    
    **Title of Report**: Brief descriptive title for this psychological inquiry
    
    **Reason for Inquiry**: Why the psychological evaluation was conducted
    
    **Recommendations**: Specific recommendations for teachers, parents, and/or the student based on your findings
    
    <Warning>
      All report fields are required. The state selection determines the student's next phase in the intervention process.
    </Warning>
  </Step>

  <Step title="Upload Supporting Documentation">
    Attach a PDF document with additional assessment results, forms, or supporting materials.
    
    - **File Type**: PDF only
    - **Maximum Size**: 5 MB (5120 KB)
    - **Optional**: Not required, but recommended for comprehensive documentation
    
    ```php
    // PsicoController.php:254-268
    if ($request->hasFile('annex_one')) {
        $file = $request->file('annex_one');
        $studentId = $id;
        $fileName = 'student_' . $studentId . '_' . now()->format('Ymd_His') . '.pdf';

        $annexPath = $file->storeAs(
            'annexes/student_' . $studentId,
            $fileName,
            'public'
        );
    }
    ```
  </Step>

  <Step title="Submit Report">
    Save the report to:
    - Update the student's demographic information
    - Change the student's state to the selected status
    - Store the psychological report with metadata
    - Save the uploaded annex (if provided)
    
    ```php
    // PsicoController.php:231-295
    DB::beginTransaction();

    try {
        $student->update([...]);
        
        $group = Group::findOrFail($request->group);
        $director = Users_teacher::where('group_director', $request->group)->first();

        Psychoorientation::create([
            'psychologist_writes'    => Auth::id(),
            'id_user_student'        => $id,
            'age_student'            => $request->age,
            'group_student'          => $group->group,
            'director_group_student' => $director
                ? $director->name . ' ' . $director->last_name
                : 'No asignado',
            'title_report'           => $request->title_report,
            'reason_inquiry'         => $request->reason_inquiry,
            'recomendations'         => $request->recomendations,
            'annex_one'              => $annexPath,
        ]);

        DB::commit();

        return redirect()
            ->route('index.student.remitted.psico')
            ->with('success', 'Informe guardado correctamente.');

    } catch (\Exception $e) {
        DB::rollBack();
        return back()->with('error', $e->getMessage());
    }
    ```
  </Step>
</Steps>

## Student History

Counselors can view the complete intervention history for any student in their assigned grades.

### Accessing Student History

```php
// PsicoController.php:301-318
public function show_student_history(string $id)
{
    $student = Users_student::findOrFail($id);

    $referrals = Referral::where('id_user_student', $id)
        ->orderBy('created_at', 'desc')
        ->paginate(10);

        ->orderBy('created_at', 'desc')
        ->paginate(10);

}
```

## Report Metadata

Each psychological report stores comprehensive metadata:

| Field | Description |
|-------|-------------|
| psychologist_writes | ID of the counselor who created the report |
| id_user_student | Student ID |
| age_student | Student's age at time of report |
| group_student | Student's group/section |
| director_group_student | Name of the group director (or "No asignado") |
| title_report | Report title |
| reason_inquiry | Reason for psychological inquiry |
| recomendations | Recommendations for intervention |
| annex_one | Path to uploaded PDF document |
| created_at | Timestamp of report creation |
| updated_at | Timestamp of last update |

## File Storage Structure

Annexes are stored in the public storage disk with organized paths:

```
storage/app/public/annexes/
└── student_{student_id}/
    ├── student_{student_id}_20260306_143022.pdf
    └── student_{student_id}_20260308_091545.pdf
```

<Note>
  Files are named with timestamps to prevent overwrites and maintain version history when multiple reports are created.
</Note>
